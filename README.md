# iss-live-tracker
 Real-time ISS (International Space Station)  tracker built with NASA's open API. Made for Hack Club Stardance.

# ISS Live Tracker 🛸

I've always been interested in space, so I thought why not build 
something around it. This tracks the International Space Station 
in real time — you can see exactly where it is on the map, its 
coordinates, and who's currently aboard.

## What it does

- Shows ISS position live on a map (updates every 5 seconds)
- Displays current latitude & longitude
- Shows the crew members currently on the ISS

## How I built it

Pretty simple stack — just HTML, CSS and JS. 
Used the wheretheiss.at API for live position data and 
Open Notify API for crew info. Leaflet.js for the map.

Got the idea from my interest in space and wanted to see 
if I could pull live satellite data into a webpage. Took 
me a bit to figure out the API integration but got there!
