# 🛰️ ISS Live Tracker

This is my ISS Live Tracker project made for Hack Club Stardance.

When I started this project, it was very simple. It only showed ISS location on map. Then I slowly learned more about APIs and JavaScript and started adding more features by myself.

Now it can show live ISS location, altitude, speed, astronauts currently in space, tracking path and some extra things which I added while learning.

## Features

- 🌍 Live ISS location
- 🛰️ Moving ISS on map
- ✨ ISS trail path
- 📍 Latitude and Longitude
- 🚀 Altitude and Speed
- 👨‍🚀 Current crew on ISS
- ⏱️ Auto refresh every 5 seconds
- 📊 Refresh countdown bar
- 📏 Distance travelled while page is open
- ⏸️ Pause and Resume tracking
- 🎯 Center map on ISS

## How I made it

I used HTML, CSS and JavaScript.

For map I used Leaflet.js.

For ISS data I used WhereTheISS API and for astronauts I used SpaceDevs API.

While making this project I learned Fetch API, async/await and how to update live data without refreshing page.

One thing which took me some time was making ISS move smoothly on map and showing its path. I also changed the UI because first version was looking very basic.
